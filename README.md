# Battleship
A web application where the user can play battleship against a computer. Users can place their ship and choose places on the "board" to "attack" the computer. This is based off something that I built using SmartBuilder. 

## Features
**User Side**
* Put ships on the board
* Select places to "attack" the computer
**Computer Side**
* Logically chooses which place to "attack" the user
* Randomly places pieces on their board

## Technologies Used
* HTML, CSS, React
* Claude.ai and ChatGPT

### How I used AI
* Getting the board to show up
* Validating user ship placement
* The ship animation for the computer loading page
